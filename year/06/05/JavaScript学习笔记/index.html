作用域是变量可以使用的范围，常见类型有—局部变量、全局变量；局部变量只能在有限的作用域内使用，它是定义在函数内部的变量，形参就是局部变量；全局变量定义在函数外部的变量，这个变量在任何函数中都有效

数组常用方法

var array = [1,3,4];
var l = array.pop(4,5); //在数组的末尾添加4，5两个元素，返回数组的长度
var b = array.pop(); //会取出数组最后一个元素，返回值是该元素

var a = [1,3,4];
var b = a.shift(); //会在数组的前面移除元素，返回该元素
console.log(a); //[3,4]
console.log(b); //1

var a = [1,3];
var c = a.unshift(12,2); //c [12,2,1,3]

var d = a.reverse(); //逆向排序，从大到小
var e = a.sort(); //升序排列，从小到大

var a = [1,2,3,4,5];
var b = a.contact(3.5); //追加数据，不会改变原数组 b ->[1,2,3,3,5]
var b = a.slice(1,3); //取出数组中指定区域的数据，不会改变原数组 b -> [2,3] 前闭后开，从1开始，不取3

var b = a.splice(1,2); //取出数组中指定区域的数据，会改变原数组 b -> [2,3] 下标从1开始，长度是2,a -> [1,4,5] 删除操作
var b = a.splice(3,0,”美队”,“复仇者”); //插入操作：必须要有三个以上的参数，第二个为0，第一个为要插入的位置，其他为要插入的元素a -> [1,2,3,”美队”,”复仇者”,4,5]
var b = a.splice(3,1,”美队”) //替换操作 a -> [1,2,3,”美队”,4,5]

var b = a.join(“-“); //把数组元素，以字符串的形式拼接join内的入参 1-2-3-4-5

JS事件

事件模型：内联模式、脚本模式

内联模式，就是酱事件写在html标签中

ps:写在window.onload 中的函数，无法被内联模式调用！

脚本模式：可以用脚本的模式调用，可以让html 和js 分离

事件类型：鼠标事件、键盘事件、HTML事件
常见的事件处理函数：on + 事件名称，比如 onclick、onload、onkeydown、ondbclick、onmouseup、onmouseover、onmouseout、onmousemove
ps:函数名都是小写字母

键盘事件：onkeydown(键盘上任意一个键按下时触发)、onkeypress(键盘上的字符键按下时触发)、onkeyup(键盘上的按键松开事件)

HTML事件小结：

onblur(); //失去焦点时触发
onsubmit(); //form节点上，点击提交按钮触发
onreset(); //点击了重置按钮触发
onscroll(); //当页面滚动时触发
onunload(); //用户离开页面时被触发
onload(); //用户页面完成加载时触发
